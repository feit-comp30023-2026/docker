# Docker
Thank you for taking the interest to explore Docker!

This semester, we have prepared materials on using Docker and Dev Containers.

These materials are intended for students who are interested in using Docker
to complete practical activities and the projects locally. VMs are still available
as an environment to complete the projects through the `VM Access` Assignment on the LMS.

Regardless of if you're using a VM or Docker, you should always inspect the CI results,
and we'd still expect your submission to execute successfully on x64 Linux
(i.e. without architecture or kernel dependent bugs).

Questions and feedback on the material are welcome.
